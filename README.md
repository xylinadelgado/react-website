# Valley Roots
> A website to help discover small and local business in the Phoenix area



## Description
Hello my name is Xylina and i was born and raised in Phoenix, arizona. I have a passionate love for my community and local buisnesses. ever since i got my first car i have been up and down and around the valley. i came up with this idea when i saw a tiktok of a local buisness having trouble in getting exposure. it broke my heart but i knew a way i could help. so i have decided to create this website and I hope to keep improving it and only get better from here.

## Features
- Google map API (currently unavalible)
- Search feature
  

## Technologies Used
- React + vite
- Mongoose/ Express
- css

## Setup/Installation
1. Clone the repo
2. Run `npm install`
3. npm run dev

## How to Use
Click on the search bar and enter what you are looking for. The filter searches by names like cafe, coffe shop, thrift, ect.

## What I Learned
During this project I learned React, Node.js, Express, and how to use API's in a project Along with that, I learned how to make a full stack website to improve my work flow.
